# agro-hub.africa-
Agro-Hub Africa : Le trait d'union entre le champ et la maison. Nous aidons les agriculteurs à vendre mieux, pour permettre aux ménages de manger mieux. une Plateforme numérique dédiée à la mise en relation des agriculteurs africains et des acheteurs locaux pour optimiser la chaîne d'approvisionnement agricole.
